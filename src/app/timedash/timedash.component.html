Timedash

BUISINESS MONITORING MADE SIMPLE

Timedash is a tool that allows employers to monitor each employee's performance.Each employee can log his hours for each project he's assigned to.After each month the employer can then asses if the employee's logs are satisfactory where he can either validate or dispute the employee's timesheet.

TECHNOLOGIES USED

  • Angular
  • Angular material
  • Firestore
  • Firebase storage
  • Firebase hosting
  • Firebase authentication
  • MongoDB
